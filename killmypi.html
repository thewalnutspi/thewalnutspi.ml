So... how do I use this?

Well, first download it:

cd /tmp
git clone https://github.com/thewalnutspi/killmypi.git
cd killmypi

Then make KillMyPi.sh executable and run it:

chmod +x KillMyPi.sh
./KillMyPi.sh

When you run it, you'll be given a warning:

pi@harry:/tmp $ ./KillMyPi.sh
To confirm you have a fire extinguisher ready - use the -justdoit argument
./KillMyPi.sh -justdoit
